# Flashcard Quiz App

## Description

This project is a Flashcard Quiz App developed as part of the CodeAlpha App Development Internship.

The application helps users study using flashcards. Users can view questions, reveal answers, move between flashcards, and manage their own study cards.

## Features

* Show Answer button
* Next and Previous navigation
* Add Flashcards
* Edit Flashcards
* Delete Flashcards
* Local Storage support

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download the project files.
2. Open the folder in VS Code.
3. Open index.html in a browser.

## Project Outcome

This project demonstrates basic frontend development and local data storage concepts.
